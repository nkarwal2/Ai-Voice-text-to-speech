# AI Voice Agent Demo (React + Node)

## Features
- Free Speech-to-Text using Browser Web Speech API
- Free Text-to-Speech using SpeechSynthesis
- Node backend with HuggingFace AI reply (free) + fallback logic
- Mock Google Calendar endpoint for demo without OAuth

## Run Backend
```bash
cd backend
npm install
npm start
```

## Run Frontend
```bash
cd frontend
npm install
npm start
```
